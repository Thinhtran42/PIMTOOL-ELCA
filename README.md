# PIMTOOL-ELCA

 - Project Information Management: Aiding in the management of projects and personnel within this project.
 - Some features:
       + View list project with some filter and sort, pagination,... Offers the ability to search for projects based on their name, status, customer, and project number.
       + Add/Modifield project with information and group, employee within this project.
- Technologies:
       + Backend: Asp.net Core API 7.0, SQL Server, Entity framework core
       + Frontend: Angular, ng-bootstrap, primeng, rxjs, ngx-bootstrap
