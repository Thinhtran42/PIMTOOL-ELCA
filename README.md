# PIMTOOL-ELCA

Project Information Management: Assisting in the management of projects and personnel within this project.

Some features:

View a list of projects with various filters, sorting options, pagination, etc. Additionally, it offers the ability to search for projects based on their name, status, customer, and project number.
Add/Modify project information, groups, and employees within this project.
Technologies:

Backend: ASP.NET Core API 7.0, SQL Server, Entity Framework Core.
Frontend: Angular, ng-bootstrap, primeng, rxjs, ngx-bootstrap.
